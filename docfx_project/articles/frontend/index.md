# Referencia Técnica del Frontend (React)

Bienvenido a la sección de documentación de la interfaz de usuario de **KeyStock**. Aquí encontrarás el detalle de los componentes, contextos y la lógica de navegación.

## ?? Estructura de Módulos
* **[Contexto de Autenticación](AuthContext/README.md)**: Gestión de sesiones y Login.
* **[Páginas (Views)](pages/README.md)**: Dashboard, Productos, Usuarios, etc.
* **[Componentes Globales](components/README.md)**: Navbar, Sidebar y rutas protegidas.

## ?? Tecnologías
El frontend está construido sobre **React 18** utilizando **Axios** para el consumo de la API de C#.
